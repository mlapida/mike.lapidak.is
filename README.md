# mike.lapidak.is

Personal site for Mike Lapidakis — Security & Networking SA Leader at AWS. Built with Astro 5, deployed to Cloudflare Pages.

## Stack

- **Astro 5** — static output, no adapter
- **Tailwind CSS v4** via `@tailwindcss/vite`
- **Cloudflare Pages** — hosting + CI/CD (auto-deploys on push to `main`)
- **Cloudflare R2** — photo storage (`assets.lapidak.is` CDN)
- **Immich** — self-hosted photo management; albums ending in `-share` sync to the site

## Development

```bash
npm install
npm run dev          # http://localhost:4321
npm run build        # production build → dist/
npm run preview      # preview built output locally
```

## Photo Sync

Photos are managed in Immich and synced automatically. To sync manually:

```bash
cp .env.example .env.local   # fill in credentials
npm run sync:dry             # dry run — no writes
npm run sync                 # sync + git commit (no push)
npm run sync:build           # sync + build
```

**Album naming convention:**

| Album name | Result |
|---|---|
| `site-share` | Singles (no collection) |
| `paris-share` | Collection "Paris" (from `Title: Paris` in album description) |

Only albums ending in `-share` are synced. Images are resized to 3200px max edge and uploaded to R2. Metadata is written to `src/content/photos/*.md`.

### Required env vars

```
IMMICH_URL=https://immich.example.com
IMMICH_API_KEY=
R2_ACCOUNT_ID=
R2_ACCESS_KEY_ID=
R2_SECRET_ACCESS_KEY=
R2_BUCKET_NAME=site-photos
PHOTOS_CDN_URL=https://cdn.example.com
SITE_REPO_PATH=.
```

See `.env.example` for a full template.

## Project Structure

```
src/
  config/site.ts              # site metadata, social links
  content/photos/*.md         # photo frontmatter (auto-generated by sync)
  layouts/Base.astro          # base HTML, OG/Twitter meta, fonts
  components/Nav.astro        # fixed nav with Dynamic Island support
  components/Footer.astro     # social links + copyright
  pages/
    index.astro               # homepage + empty.coffee RSS
    photography.astro         # photo grid + collection cards
    photography/[slug].astro  # per-photo detail: EXIF, Leaflet map, prev/next
    photography/collection/[slug].astro  # per-collection grid
    work.astro                # professional background
  styles/global.css           # CSS custom properties
scripts/
  sync-immich.mjs             # Immich → R2 + .md sync script
  setup-sync.sh               # one-time Docker sidecar setup
public/
  favicon-16x16.png
  favicon-32x32.png
  favicon.ico
  apple-touch-icon.png
  social-card.jpg
  profile.jpg
  robots.txt
```
